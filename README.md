This project presents a comparative analysis of YOLOv5 and YOLOv6 object detection models.

The objective of the study is to evaluate the performance of both models for object detection tasks by comparing their accuracy, inference speed, and computational complexity.

The models were trained and tested in a Google Colab environment using a benchmark dataset. Performance metrics such as mean Average Precision (mAP), inference time, and GFLOPs were analyzed to understand the trade-offs between detection accuracy and computational efficiency.

The results are visualized using comparison graphs and tables to highlight differences between the two architectures.

Evaluation Metrics:
- Mean Average Precision (mAP)
- Inference Time
- GFLOPs

Tools and Technologies:
- Python
- PyTorch
- YOLOv5
- YOLOv6
- Google Colab
- Matplotlib
- Pandas
